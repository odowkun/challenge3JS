Soal :
Diberikan sebuah string yang hanya terdiri dari huruf.outputkan panjang setiap substring yang memiliki huruf yang sama.
upperCase dan lowerCase akan dihitung sama.
// Contoh input:
// aaabbccCAAaaC
// output:
// Lowercase == Uppercase
// a = 7
// b = 2
// c = 4
// Lowercase != Uppercase
// a = 5
// b = 2
// c = 2
// C = 2
// A = 2
